IMAGE GALLERY
A responsive image gallery built uding HTML, CSS , and JAVASCRIPT with category filtering ,hover effects, lightbox preview,and image navigation.

OVERVIEW

This project is a responsive image gallery developed using HTML, CSS, and JavaScript. It allows users to view images in a grid layout, filter images by category, and preview images in a lightbox with navigation controls.

FEATURES

- Responsive grid layout
- Category-based image filtering
- Lightbox image preview
- Previous and Next navigation
- Hover zoom effects
- Mobile-friendly design
- Smooth transitions and animations

TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript

1. Download or clone the repository.
